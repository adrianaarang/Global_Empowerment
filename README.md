# The Global Empowerment — Web Corporativa

ONG de cooperación internacional enfocada en el empoderamiento de mujeres rurales.

## Stack
- Frontend: HTML5 + CSS3 + JavaScript vanilla
- Backend: PHP 8+
- Base de datos: MySQL
- Hosting: AWS Lightsail

## Instalación
1. Copiar `.env.example` a `.env` y rellenar credenciales
2. Ejecutar `database/schema.sql` en MySQL
3. Apuntar el servidor web a `/public`
